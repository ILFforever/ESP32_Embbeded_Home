Implementation Notes:

  • Touch Targets: All cards and buttons should be at least 60√ó60px for reliable touch input
  • Libraries: Use TFT_eSPI or LovyanGFX for rendering with hardware acceleration
  • Performance: Update only changed card areas (partial refresh) to maintain smooth UI
  • Colors: Use 16-bit RGB565 color format for ESP32 compatibility
  • Animations: Keep animations minimal - simple fade or slide transitions work best
  • Double Buffering: Use PSRAM for smooth rendering on larger displays
